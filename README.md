# Ipl-match-data-analysis
Project Overview

This project presents an interactive Power BI dashboard built to analyze Indian Premier League (IPL) match data. The analysis focuses on match outcomes, team performance, toss impact, player contributions, and venue-wise trends across multiple IPL seasons.

The dataset consists of 1000 IPL match records containing match-level details such as teams, scores, winners, toss decisions, player performances, and umpire information. Using Power BI, the data was cleaned, transformed, and visualized to extract meaningful insights and trends.

📁 Dataset Description
File Name: IPL_Match_Analysis_1000_Records_v2.xlsx
Total Records: 1000 matches
Total Columns: 21
Data Type: Structured match-level IPL data

📌 Key Columns Used
Match & Season Details
Match_ID, Match_Date, Season
Location Details
Venue, City
Team & Toss Information
Team1, Team2, Toss_Winner, Toss_Decision
Match Scores
First_Innings_Score, Second_Innings_Score
Match Outcome
Winner, Margin, Match_Result
Player Performance
Player_of_Match, Top_Scorer, Runs_Scored
Top_Wicket_Taker, Wickets_Taken
Officials
Umpire1, Umpire2

🎯 Project Objectives
Analyze IPL match results across seasons
Understand the impact of toss decisions on match outcomes
Compare team performances and winning trends
Identify top-performing players (batting & bowling)
Analyze venue and city-wise match distribution
Create an interactive and visually appealing Power BI dashboard
